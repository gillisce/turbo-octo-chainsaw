## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Get lasted version of NPM
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

```
git clone https://github.com/gillisce/turbo-octo-chainsaw.git my-app

cd my-app

npm install
```
